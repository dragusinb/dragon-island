Level Design Audit

Every action needs visual feedback

"Use a level designer agent — every action needs visual feedback. E.g. I see apples in trees."

Expert audited all 16 interactable types

Stone nodes: invisible mining → Visible cracks + particle burst
Fishing spots: underground! → Raised to water surface + ripple effect
Mushrooms: vanish completely → Shrink animation + spore particles
Trees: no stump after chop → Stump remains + fall animation
Herbs: invisible flowers → Visible bloom + petal scatter on pick

Iterative Polish: User Testing Loop

Real-time testing → immediate fixes — each under 2 minutes

"I can't see apples" → Apples were inside canopy mesh — moved outside
"Wood trees look same as apple trees" → Changed to fir/pine shape (stacked cones)
"Moon goes through the map" → Orbit radius 200 → 900
"Artefacts are hard to find" → Added golden light beams shooting into sky
"Where's my inventory? Controls?" → Added H=Help, M=Map, artefact journal
Plain language bug report → deployed fix in < 2 minutes
